- 制限時間:3時間コース 
- 使用言語:Ruby

以下のレギュレーションを満たすように実装を行ってください

1. WEBサイトのルートURLへ接続すると Hello World と表示される
1. HerokuボタンからHerokuに対してデプロイすることができる
